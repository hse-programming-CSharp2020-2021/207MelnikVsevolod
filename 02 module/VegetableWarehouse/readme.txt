	Взаимодействие с программой происходит при помощи команд,
описание которых можно узнать вызвав команду help.

	Чтобы вывести информацию о складе - вызовите команду info
или export, если вы хотите вывести информацию в файл.
	Чтобы проверить работу файлов вы можете использовать готовые
файлы wh.txt, cmd.txt и containers.txt, которые находятся в папке test.

	Дополнительный функционал - возможность назвать ящики и 
найти их по наименованию.
